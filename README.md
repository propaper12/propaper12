I build systems that bridge the gap between raw data and intelligent applications. My focus is on **Separation of Concerns, Medallion Architecture (Bronze/Silver/Gold), and Self-Healing Data Pipelines.** I don't just move data; I engineer environments where machine learning models can autonomously train, evaluate, and deploy in real-time.

## 📊 Infrastructure & Telemetry Capabilities

I don't just write code; I orchestrate robust infrastructures. My environments typically feature:
* **Distributed Control Planes:** Using Redis for in-memory auth, distributed rate-limiting, and microservice sync.
* **Observability:** Complete telemetry stacks with **Prometheus** and **Grafana** tracking hardware and container metrics (cAdvisor).
* **Self-Healing Mechanics:** Automated data optimization and vacuuming in Delta Lake to manage small-file problems and storage leaks.
