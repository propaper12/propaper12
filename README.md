## 🚀 About Me

I'm a **Data Engineer and MLOps Architect** with a knack for transforming complex data challenges into streamlined, self-healing solutions. With deep expertise in data pipelines and machine learning, I thrive on building real-time streaming systems and scalable Lakehouse architectures. 

My goal is to build systems that don't just move data, but create environments where AI models can autonomously train, evaluate, and deploy in real-time. From raw ingestion to AI-ready datasets, I focus on scalable, resilient architectures.

- 🏢 **Currently:** Architecting real-time streaming systems and studying Management Information Systems (YBS) at **Istanbul Gelişim University**.
- 🌱 **Exploring:** Advanced RAG (Retrieval-Augmented Generation), AI Agents, and Medallion Architecture optimization.
- 💬 **Ask me about:** Data pipelines, machine learning models, sub-5 second latency architectures, and distributed systems.
- ⚡ **Fun fact:** I treat my code like a high-frequency trading bot—every millisecond and every megabyte of RAM matters!
