### Hi there 👋

I'm a **Data Engineer and MLOps Architect** focused on building enterprise-grade Data-as-a-Service (DaaS) platforms, ultra-low latency streaming pipelines, and autonomous AI-driven data ecosystems. I help ensure raw data is transformed into actionable intelligence using Medallion Architectures and self-healing systems. 

I'm extremely passionate about bridging Big Data strategy with hands-on machine learning depth. My goal is to build systems that don't just move data, but create environments where AI models can autonomously train, evaluate, and deploy in real-time. Whether it's processing high-frequency financial data or integrating LLMs and RAG systems into data workflows, I focus on scalable, resilient architectures.
